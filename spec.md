# ImageForNet

## 技術仕様と実装報告

### 背景と目的

#### 課題

ネット上では画像ファイルアップロードする時、EXIFを除去、透かしテキストの追加をしたいが、
マルチプラットフォームを対応したものを発見する事ができない。
１つのサイトを異なるOSを所有しているメンバーが同一フォーマットで画像加工する事ができない

#### 目的

マルチプラットフォームを対応することにより、異なるOSを所有しているメンバーが
同一フォーマットで画像加工する事ができる

#### システム概要

* 機能:
  * EXIFを除去
  * 透かしテキスト（ウォーターマーク）の追加
    * 設定内容：位置（四隅）、透かし文字、色（RGBA）、フォントサイズ、不透明度
  * プレビュー表示
    * 画面右側に加工後の状態をリアルタイム（または設定変更時）に確認できるプレビューエリアを実装
    * 起動時にサンプル画像を表示し、加工イメージを即座に把握可能
  * スマートなフォルダ選択
    * 初回起動時はピクチャーフォルダを参照（Linuxのみ。それ以外のOSや取得不可時はホームフォルダを参照）
    * 以降、最後に開いたフォルダや保存したフォルダを記憶し、ユーザーの利便性を向上
* 環境
  * Avalonia
  * .NET 8

* 実装
  * 画像ライブラリ
    * SixLabors.ImageSharp
    機能と環境を網羅する
  * MVVM フレームワーク
    CommunityToolkit.Mvvm: .NET 8 と相性
